komtet_kassa_joomla
===============================

Чтобы развернуть окружение и потестировать плагин в docker контейнере:
1. создать в дирректории docker_env папку /php, скопировать в нее текущий код CMS Joomla 3
2. создать папку для хранения данных БД /data/mysql
3. выполнить docker-compose up -d
4. перейти в браузере на localhost:8100 и выполнить установку CMS, параметры подключения к БД указать
    host: mysql,
    user: root,
    password: my_secret_pw_shh,
    db: test_db
5. установить и настроить плагин через инсталлер
